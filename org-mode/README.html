2 Export

File org mode dapat di-export ke berbagai jenis dokumen, misalnya ke pdf (via latex), html, dsb.

2.1 HTML

command-nya adalah: 

c+c c+e

Kemudian pilihlah jenis dokumen yang diinginkan dari menu yang muncul.

3 CSS

Download bluesimple.css

Tambahkan command berikut pada bagian atas file.
